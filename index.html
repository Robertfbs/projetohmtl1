Entendendo Comunicação Client x Server

Neste módulo da aula, aprendemos os dois lados da "força" existentes, o dos Jedi e o dos Sith. Bem, não são exatamente estes dois lados, mas é quase isso. Aprendemos que existe o lado "Client" e o lado "Server", que basicamente são, o lado do cliente e o lado do servidor, sendo "cliente" o usuário final, como num caso de página na web, o cliente seria a pessoa que acessa este site e o Server, (servidor) seria basicamente um PC, que mantém as informações deste site armazenadas para que quando o cliente navegue para buscá-la, ela esteja sempre em mãos. Aprendemos de forma básica, como é a comunicação entre esses dois lados e algumas aplicações utilizadas em um deles ou até mesmo nos dois, como é o caso do javascript.

Introdução ao HTML na prática!

Neste módulo, descobrimos as principais ferramentas utilizadas para que possamos começar a codar e aprender mais sobre HTML, como o Visual studio Code, sendo nossa principal ferramenta para codar, não sendo a única existente, porém a mais utilizada pela galera da DIO.

Descobrimos o inspetor de elementos que nos oferece a chance de verificar como foi codado aquela determinada página que estamos visualizando, sendo uma ótima ferramenta pois nos permite verificar como aquele site que tanto achamos irado foi codado e as principais tags e atributos utilizados para construção dele.

Por falar em tags e atributos, aprendemos também o que são tags e atributos e algumas das tags e atributos mais utilizados para construção de uma página bem estruturada em HTML, onde "Tags" são basicamente estruturas que contém algum tipo de instrução a ser interpretada pelo navegador. Elas possuem uma marcação para início e uma para fim da tag, determinando que as alterações geradas por elas, só funcionem no que está preenchido dentro destas marcas. Um exemplo simples, seria a tag "i" que serve para deixar um texto em itálico, desta forma. E atributos por sua vez, são estruturas que dão algum valor a uma determinada tag, como por exemplo os atriutos "Text" e "Numbers" da tag "input", A tag "Input" serve para abrir uma caixa para preenchimento, como aquelas famosas em formulários onde pedem para preencher seus dados como, nome, telefone, e-mail e afins, a tag "input" é quem cria estes campos para digitação e os atributos "text" e "number" definirão o que poderá ser preenchido, se texto ou se números, então atributos alteram valores das tags, porém, não alteram sua função principal.
